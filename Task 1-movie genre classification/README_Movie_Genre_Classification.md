# Movie Genre Classification

## 📌 Project Overview

This project predicts the genre of a movie based on its plot description
using Machine Learning and Natural Language Processing (NLP).

## 🚀 Features

-   Text preprocessing
-   TF-IDF feature extraction
-   Genre prediction
-   Model training and evaluation

## 🛠️ Technologies Used

-   Python
-   Pandas
-   NumPy
-   Scikit-learn
-   NLTK

## 📂 Project Structure

``` text
Movie_Genre_Classification/
│── movie_genre.py
│── description.txt
│── README.md
└── Genre Classification Dataset/
```

## 📊 Dataset

The dataset files are too large to upload to GitHub.

Download the dataset from:
https://www.kaggle.com/datasets/hijest/genre-classification-dataset-imdb

After downloading, place these files inside the dataset folder: -
train_data.txt - test_data.txt - test_data_solution.txt

## ▶️ How to Run

1.  Install the required libraries:

``` bash
pip install pandas numpy scikit-learn nltk
```

2.  Run the project:

``` bash
python movie_genre.py
```

## 📈 Output

The model predicts the movie genre based on the given movie description.
