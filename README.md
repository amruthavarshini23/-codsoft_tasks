# CodSoft Machine Learning Internship

Welcome to my CodSoft Machine Learning Internship repository.

This repository contains the Machine Learning projects I completed during my internship.

## 📁 Projects

### Task 1 – Movie Genre Classification
- Predicts movie genres from movie plot descriptions using Machine Learning and NLP.

### Task 2 – Credit Card Fraud Detection
- Detects fraudulent credit card transactions using Machine Learning.

### Task 3 – Customer Churn Prediction
- Predicts whether a telecom customer is likely to leave the service.

### Task 4 – Spam SMS Detection
- Classifies SMS messages as Spam or Ham using Natural Language Processing.
