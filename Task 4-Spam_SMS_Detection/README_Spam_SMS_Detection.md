# Spam SMS Detection

## 📌 Project Overview

This project uses Machine Learning and Natural Language Processing (NLP)
to classify SMS messages as **Spam** or **Ham (Legitimate)**.

## 🚀 Features

-   SMS text preprocessing
-   TF-IDF vectorization
-   Multinomial Naive Bayes classifier
-   Interactive prediction for custom SMS messages

## 🛠️ Technologies Used

-   Python
-   Pandas
-   Scikit-learn
-   NumPy

## 📂 Project Structure

``` text
Spam_SMS_Detection/
│── spam_sms_detection.py
│── spam.csv
│── README.md
```

## ▶️ How to Run

``` bash
pip install pandas numpy scikit-learn
python spam_sms_detection.py
```

## 📊 Dataset

The project uses the `spam.csv` dataset containing labeled SMS messages.

## 📈 Output

The model classifies each message as: - 🚨 Spam - ✅ Legitimate (Ham)
