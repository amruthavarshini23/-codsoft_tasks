# Customer Churn Prediction

## 📌 Project Overview

This project predicts whether a customer is likely to leave a telecom
service using Machine Learning classification models.

## 🚀 Features

-   Data preprocessing
-   Label encoding
-   Logistic Regression
-   Random Forest
-   Gradient Boosting
-   Confusion Matrix and Feature Importance visualization

## 🛠️ Technologies Used

-   Python
-   Pandas
-   NumPy
-   Scikit-learn
-   Matplotlib
-   Seaborn

## 📂 Project Structure

``` text
Customer_Churn_Prediction/
│── customer_churn_prediction.py
│── WA_Fn-UseC_-Telco-Customer-Churn.csv
│── README.md
```

## ▶️ How to Run

``` bash
pip install pandas numpy scikit-learn matplotlib seaborn
python customer_churn_prediction.py
```

## 📊 Dataset

The project uses the Telco Customer Churn dataset
(`WA_Fn-UseC_-Telco-Customer-Churn.csv`).

## 📈 Output

The project compares multiple machine learning models and predicts
whether a customer will stay or churn.
